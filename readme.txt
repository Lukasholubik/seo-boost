Audit Dashboard, Redirect Manager a další SEO nástroje doplňující Rank Math (Free).

== Popis ==

SEO Booster Pro přidává do WordPressu:

* SEO Audit Dashboard – hromadný scan všech stránek, skóre, inline opravy
* Redirect Manager + 404 log
* SERP náhled s pixelovou šířkou
* (plánováno) Interní prolinkování, Content Decay Monitor, Search Console integrace, Local SEO, Image SEO Optimizer, white-label reporty

Plugin je navržen tak, aby nekolidoval s Rank Math (Free) – čte a zapisuje hodnoty title/description přes Rank Math meta klíče.

== Instalace ==

1. Nahrajte složku `seo-boost` do `/wp-content/plugins/`
2. Aktivujte plugin přes menu Pluginy → Aktivovat
3. Nastavení najdete v menu „SEO Booster"
